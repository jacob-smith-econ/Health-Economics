#Overview
This is a set of slides I created for my MSMG5450H class at Trent University. The objective of these slides was to give a general overview of the models and theories  used in health economics with the first half focusing on what I considered the "core" theoretical models in health economics and the second half discussing tools employed by those in health technology assessment (HTA) or pharmacoeconomics. Care was taken to teach the material without any calculus (though there is some skimming of results in lecture #6).

There may be some typos in this current set of slides . Hopefully this will be corrected in future iterations.
